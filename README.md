# CLV-Custome-Lifetime-Value-
📊 Customer Lifetime Value (CLV) Dashboard

Welcome to my Customer Lifetime Value (CLV) project! This repository showcases how to calculate, analyze, and visualize CLV using SQL, Python, and Power BI. The goal is to understand customer behavior, predict long-term value, and create actionable insights for business growth.

🚀 Project Overview

Objective: Estimate the lifetime value of customers based on purchase history and engagement.

Tech Stack: SQL, Python (Pandas, Matplotlib, Scikit-learn), Power BI.

Output: Interactive Power BI dashboard with KPIs and visualizations.

📂 Repository Structure

data/ → Raw and cleaned datasets.

sql/ → SQL scripts for data extraction and preprocessing.

python/ → Python notebooks for CLV calculation and modeling.

dashboard/ → Power BI files and screenshots.

docs/ → Documentation and references.

📈 Key Features

CLV Calculation: Using historical transaction data.

Segmentation: Grouping customers by value tiers.

KPIs:

Customer Pulse

Revenue Radar

Churn Buster

Loyalty Ladder

Growth Gauge

Visualizations:

Stacked Bar Chart → Performance Breakdown

Ribbon Chart → Rank Trend

Pie Chart → Customer Share

Donut Chart → Category Share

Funnel Chart → Conversion Flow
